# Objective
    Write a function that takes in a string and number that repeats the string that number of times.

# Example Returns:
* repeating('codingdude', 2) will return **'codingdudecodingdude'**.
* repeating('google', 3) will return **'googlegooglegoogle'**.


# How I Solved It:
* repeat(), push(), join()
* while loop
