# Objective
    Write a function that takes in a string and returns the largest word in the array.

# Example Returns:
* longestWord('Ready, steady, go!') will return **'steady'**.
* longestWord('Ready[[[, steady, go!') will return **'steady'**.
* longestWord('ABCd') will return **'ABCd'**.

# How I Solved It:
* split(), replace()
* for loop and if statement
